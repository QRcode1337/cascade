## Program lanes (what to use when)

### Lane 1: No website → “Digital Receptionist Drop”
**Best for:** local services, walk-ins, “missed calls,” businesses with no online booking.  
**Use:**
- `sales/no-site-outreachpack.md`
- `sales/high-impact-outreach-scripts.txt`
- `program/demo-setup-guide.md` (Demo #1 / #2)
- `../playbooks/typebot/flow-templates.json`
- `../playbooks/make/automation-blueprints.md`

### Lane 2: Outdated website → “Digital Employee Overlay”
**Best for:** contractors, clinics, firms with a site that exists but does nothing.  
**Use:**
- `program/lane-2-strat-guide.txt`
- `sales/website-improvement-email-templates-contractors.md`
- `program/demo-setup-guide.md` (Demo #2 / #3)
- `../playbooks/make/automation-blueprints.md`

### Lane 3: Restaurants → “Reputation & Reservation Engine”
**Best for:** restaurants, bars, high-volume appointment businesses.  
**Use:**
- `program/lane-2-strat-guide.txt` (restaurant hooks are in here)
- `../playbooks/comms/email-sms-templates.md`
- `../playbooks/make/automation-blueprints.md`
